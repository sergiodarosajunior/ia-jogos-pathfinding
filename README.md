## Projetos

### PathfindingFramework

Projeto em Godot 4 com a implementação de algoritmos de busca em grafos para pathfinding:

- **BFS** (Breadth-First Search) - Busca em largura usando fila FIFO
- **Dijkstra** - Busca por custo acumulado usando fila de prioridade
- **Greedy** (Busca Gulosa) - Busca por heurística (distância Manhattan) usando fila de prioridade
- **A*** - Combinação de custo acumulado + heurística (f = g + h)

**Como rodar:** Abrir o projeto `PathfindingFramework/path-finder-framework/` no Godot 4 e executar a cena principal.
